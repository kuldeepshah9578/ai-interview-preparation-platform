## Overview

The **AI-Powered Interview Preparation Platform** is a full-stack web application designed to help users practice technical interview questions and receive instant evaluation and feedback.

This project simulates a real-world interview system by generating answers, scoring responses, storing performance history, and providing analytics dashboards. It demonstrates production-ready backend development, REST API design, database integration, and cloud deployment.

The application is deployed live on the internet using **Render**, making it accessible to recruiters and hiring managers.

## Core APIs

Create Interview Question:

POST /api/v1/ask

Get Interview History:

GET /api/v1/history

Get Performance Summary:

GET /api/v1/summary

Get Recommended Topic:

GET /api/v1/recommendation

Get Leaderboard:

GET /api/v1/leaderboard

Export Interview Data:

GET /api/v1/export/csv
